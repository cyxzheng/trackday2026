This project is the static `yvrgrc.ca` website.

Rules:

- Use only plain HTML, CSS, and vanilla JavaScript
- No frameworks
- No build step
- Keep each site section self-contained under its own directory when possible
- Root `index.html` should stay a simple landing page until a full homepage is intentionally built
- Keep Track Day 2026 content under `trackday2026/`
- Keep announcements content under `announcements/`
- Keep editable event content centralized and obvious in `trackday2026/script.js`
